# 取扱説明書 (Knowledge Base User Manual)

## 1. ログイン
本システムは Microsoft SSO (Azure AD) または 登録済みのメールアドレス・パスワードでログイン可能です。
- 初回ログイン時は、管理者の承認済みメールアドレスである必要があります。

## 2. ナレッジを探す
### 2.1 キーワード検索
画面左側のサイドバー上部にある検索ボックスにキーワードを入力すると、タイトルや内容に含まれる情報をリアルタイムで絞り込みます。

### 2.2 フィルタリング
- **カテゴリ**: 画面中央上部またはサイドバーから特定のカテゴリを選択できます。
- **現象 (Phenomena)**: 特定の不具合事象（現象）に絞って検索できます。
- **解決状況**: 「未解決」のみ、「解決済み」のみを表示できます。
- **マイ投稿**: 自分が作成したナレッジのみを表示します。

### 2.3 AIチャット検索
画面右下のチャットアイコンをクリックし、自然言語（例：「エルボの交換手順を教えて」「去年の水漏れ事例は？」）で質問してください。AIがナレッジベースから最適な回答を提示し、関連する記事へのリンクを表示します。

## 3. ナレッジを登録・編集する
### 3.1 新規登録
サイドバーの「+ 新規作成」ボタン、または画面上の「投稿する」ボタンから作成画面を開きます。
- **必須項目**: タイトル、号機、カテゴリ。
- **内容**: Markdown形式で詳細に記述できます。画像を貼り付けることも可能です。

### 3.2 編集・更新
ナレッジリストから記事を選択し、右上の「編集」ボタンから内容を更新できます。
- 更新すると、編集履歴が記録されます。

## 4. 管理者向け操作 (マスタ管理)
権限を持つユーザーは、画面上部の「マスタ管理」から以下の操作が可能です。
- **マスタ追加**: 新しいカテゴリや現象（現象）を追加・削除。
- **ユーザー管理**: 
  - ユーザーの追加：メールアドレスと名前を指定して事前に登録。
  - 権限変更：ドロップダウンから各ユーザーの権限レベルを選択。
- **反映**: 変更後は右上の「変更を保存」ボタンを必ず押してください。

## 5. よくある質問 (FAQ)
- **Q: 自分の投稿した記事が検索に引っかからない。**
  A: フィルタ（未解決/解決済みなど）が「すべて」になっているか確認してください。
- **Q: 権限が足りないため編集できない。**
  A: 管理者（Manager/Master権限者）に権限変更を依頼してください。
